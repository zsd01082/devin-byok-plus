📖 使用教程 必读
快速使用
  1. 第一步：配置连接 - 在「⚙️ 配置连接」页分别为 BYOK #1 / #2 填写 Base URL、API Key，加载模型并选择模型；Claude/GPT 可设置思考强度。
  2. 第二步：安装补丁 - 切换到「🔧 系统补丁」页，点击「安装补丁」将配置注入 Windsurf。
  3. 第三步：启动代理 - 切换到「🚀 控制状态」页，点击「一键启动」，补丁就绪后重载窗口；Windsurf 里分别使用 Claude Opus 4 BYOK 与 Claude Opus 4 Thinking BYOK。
日常使用
  • 只换 API Key 或模型：修改后会自动保存。
  • 聊天没有走代理：重新安装补丁并重载窗口。
  • 模型列表加载失败：检查 API Key、余额、网络和日志错误。
BYOK #1 对应 Windsurf 的 Claude Opus 4 BYOK；BYOK #2 对应 Claude Opus 4 Thinking BYOK。两套 API / 模型完全独立。
